# Valkyrie Studio

Valkyrie Studio es una plataforma para crear, gestionar y automatizar personajes virtuales impulsados por IA.

## Objetivos

- Gestión de personajes
- Biblioteca de escenarios
- Gestión de outfits
- Generador de prompts
- Generación de imágenes
- Generación de videos
- Gestión de publicaciones
- Analíticas

## Arquitectura

Frontend:
- Next.js

Backend:
- FastAPI

Base de datos:
- PostgreSQL

Infraestructura:
- Docker
